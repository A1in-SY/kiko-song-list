# vup 歌单页(ameki)

## 部署使用

### 制作歌单内容

1. 按照模板 `scrips/music.xlsx` 填写，制作歌单内容
2. 运行 `scripts/converter.exe` 生成歌单文件


### 启动开发环境

```bash
npm install
npm run dev
```

### 导出静态网站

已部署在vercel，提交代码自动构建
